Tip 01 — Use Tonight's Lineups Tab

The Lineups tab shows the actual batting order vs tonight's starter. Target batters with matchup score ≥ 70 for HR props.

Tip 02 — Elite Thresholds

🎯 Elite: .300 ISO / .500 SLG / 92+ EV / 20%+ Barrel
✅ Favorable: .200 ISO / .400 SLG / 89+ EV / 12%+ Barrel

Tip 03 — Parks + Weather

Target HR-friendly parks (🔥 1.15x+). Matchup score ≥ 70 + park ≥ 1.15x + vulnerable pitcher = triple signal.

Tip 04 — Parlay 2–3 Legs Max

Best combo: 1 top-tier batter + 1 value hitter vs same pitcher. Or: 2 teammates vs same vulnerable starter.

Tip 05 — Fade Pitcher HR Allowed

Some books offer Over 1.5 HR Allowed. Great value when multiple batters show ≥ 70 vs same starter in a hitter-friendly park.

Tip 06 — Reverse Splits + Velo

Always check HR Risk RHB vs LHB separately. Recent velo below career avg = elevated HR risk regardless of ERA.

🧠 Final Tips
Pair 2+ signals — Matchup score + park + day/night all pointing same direction.
Park amplifies — Same matchup worth more at GABP than Oracle Park.
Day/Night matters — Large gap batter in preferred slot + Matchup ≥ 70 = strong edge.
Near HR data — Cross reference PropFinder near HR counts alongside matchup scores.
