HR Prop Framework

Daily Top Picks Analytics — minimum 2 signals before any bet

Stack Vulnerable Pitchers

Use Lineups tab. Target pitchers where 2+ batters score ≥ 70. Stack teammates vs same starter.

Elite Thresholds

🎯 Elite: .300 ISO / .500 SLG / 92+ EV / 20%+ Barrel
✅ Target: .200 ISO / .400 SLG / 89+ EV / 12%+ Barrel

Park + Weather Stack

Matchup ≥ 70 + park ≥ 1.15x + wind out = triple signal. Best plays of the day.

2-3 Leg Parlays Only

1 elite + 1 value vs same pitcher. Or 2 teammates vs vulnerable starter. Never more than 3 legs.

Pitcher HR Allowed Props

Over 1.5 HR Allowed lines — great value when 3+ batters show ≥ 70 vs same starter in hitter park.

Velocity Drops = Edge

Pitcher down 2+ mph from season avg = significant HR risk increase regardless of ERA. Always check Velo Trend column.

Signal Checklist — Need 2+ Before Betting
✅ Barrel% ≥ 12% Favorable / ≥ 20% Elite
✅ Exit Velo ≥ 89 mph / ≥ 92 mph Elite
✅ HR Risk vs batter hand ≥ 1.5
✅ Pitcher Barrel% Against ≥ 7%
✅ Platoon advantage confirmed
✅ Park factor ≥ 1.10x
✅ Matchup score ≥ 70
✅ Batter in preferred day/night slot
